# Zenit Guardian 🛡️

Лёгкий «привратник» сайта ООО Зенит. Стоит **отдельным контейнером** перед основным
ботом и держит публичный порт **3000 всегда**. Поэтому при редеплое, падении, OOM или
ручном выключении бота посетитель видит красивую страницу «Сайт временно недоступен»
с авто-перезагрузкой, а **не** «Bad Gateway».

```
домен ──→ [Guardian :3000]  ──проксирует──→  UPSTREAM_URL (контейнер бота)
                 │
                 └─ бот недоступен → guardian_maintenance.html (сам перезагрузится, когда бот оживёт)
```

Пока бот жив — guardian прозрачно проксирует все запросы, поведение для пользователей
не меняется. Зависимостей нет: только стандартная библиотека Python.

## Зачем отдельный контейнер
Если guardian живёт внутри контейнера бота, то редеплой бота пересоздаёт контейнер
целиком — guardian умирает вместе с ним, и снова появляется «Bad Gateway». Отдельный
контейнер, который ты никогда не редеплоишь, решает это.

## Переменные окружения
| Переменная        | По умолчанию | Описание |
|-------------------|--------------|----------|
| `PORT`            | `3000`       | Публичный порт, который проксирует Bothost. |
| `UPSTREAM_URL`    | —            | **Обязательно.** Адрес контейнера бота, напр. `https://zenit-bot.bothost.ru`. Без него guardian всегда показывает заглушку. |
| `GUARDIAN_TIMEOUT`| `30`         | Таймаут запроса к боту, сек. |

## Настройка на Bothost
1. Залить этот репозиторий в **отдельный** бот/слот на Bothost.
2. **Домен сайта** привязать к этому слоту (guardian), а **не** к слоту бота.
3. В переменных окружения guardian указать `UPSTREAM_URL` — публичный адрес слота бота
   (его собственный URL на Bothost, на который раньше смотрел домен).
4. Готово: теперь домен идёт через guardian. Бот можно редеплоить/гасить сколько угодно —
   сайт не упадёт с «Bad Gateway».

## Локальный запуск
```bash
UPSTREAM_URL=http://127.0.0.1:8099 PORT=3000 python guardian.py
```

## Служебные ручки
- `GET /__guardian/health` — `{"up": true|false}`, доступен ли бот (использует страница для авто-перезагрузки).
- `GET /__guardian/logo.png`, `GET /__guardian/favicon.png` — логотип/иконка (guardian отдаёт их сам, чтобы заглушка не зависела от бота).
